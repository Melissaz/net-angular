## Description

MediatorApiExample is a Minimal API project built with .NET 6

## Prerequisites
.NET 6 SDK
An IDE like Visual Studio Code, Visual Studio, or Rider

## Getting Started

### 1. Restore Dependencies

dotnet restore

### 2. Build the Project

dotnet build

### 3. Run the Project

dotnet run

##  Using the API

https://localhost:7205/swagger
